## 安装

通过Composer安装,运行以下命令:

``` bash
composer require gkr/support
```

### 注册服务提供者

Next add the following service provider in `config/app.php`.

``` php
'providers' => [
  Gkr\Support\SupportServiceProvider::class,
],
```
